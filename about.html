Sobre mí

Soy Andrés Felipe Salazar, Ingeniero en Sistemas con una tecnología en Desarrollo de Software, apasionado por la inteligencia artificial y la automatización. Cuento con una sólida base en arquitectura de software, diseño de sistemas y metodologías ágiles como Scrum, lo que me permite desarrollar soluciones escalables y orientadas a resultados.

Manejo Python para backend y la integración de modelos de machine learning, además de Firebase para gestión de datos en la nube. En frontend, trabajo con Angular y React para crear interfaces modernas y funcionales. También tengo experiencia con bases de datos SQL y NoSQL, integración de APIs y arquitecturas modulares que optimizan el desarrollo y mantenimiento de aplicaciones.
